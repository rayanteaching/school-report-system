## توضیح کار

در این Pull Request چه کاری انجام داده‌اید؟

مثال:

```text
در این Pull Request ظاهر Dashboard اصلی برنامه بهتر شده است.
```

---

## Issue مرتبط

شماره Issue را بنویسید.

مثال:

```text
Related Issue: #21
```

یا اگر این Pull Request بعد از Merge باید Issue را ببندد:

```text
Closes #21
```

---

## فایل‌های تغییر کرده

فایل‌هایی که تغییر داده‌اید را بنویسید:

*
*

مثال:

* `main.py`

## موارد تغییر یافته

دقیق بنویسید چه چیزهایی اضافه یا اصلاح شده‌اند:

*
*
*

مثال:
 
* اکنون Header برنامه ایجاد شد.
* دکمه‌های Dashboard اضافه شدند.
* نوار وضعیت پایین صفحه ایجاد شد.
* تابع `clear_screen` اضافه شد.

---

## نتیجه تست

قبل از ساخت Pull Request باید برنامه را اجرا کرده باشید.

مواردی را که تست کرده‌اید علامت بزنید:

* [ ] برنامه بدون خطا اجرا شد.
* [ ] خروجی مورد انتظار را دیدم.
* [ ] فایل مربوط به کار خودم را بررسی کردم.
* [ ] برنامه را بعد از بستن، دوباره اجرا کردم.
* [ ] اگر رابط گرافیکی تغییر کرده، ظاهر آن را بررسی کردم.

توضیح تست:

```text
برنامه اجرا شد و پنجره بدون خطا باز شد.
```
