# School Report System

این پروژه یک تمرین گروهی برای یادگیری Python، Tkinter، Git و GitHub است.

در این پروژه، یک سیستم ساده برای مدیریت نمرات و نمایش کارنامه دانش‌آموزان ساخته می‌شود.

## هدف پروژه

هدف اصلی پروژه فقط نوشتن کد نیست. هدف این است که اعضای تیم یاد بگیرند چگونه مانند یک تیم واقعی نرم‌افزاری کار کنند.

در این پروژه یاد می‌گیریم:

* ساخت Repository
* ساخت Issue
* ساخت Branch
* نوشتن کد در فایل جداگانه
* Commit کردن تغییرات
* Push کردن Branch
* ساخت Pull Request
* Code Review
* اصلاح Pull Request
* Merge کردن تغییرات
* مدیریت Conflict

## ساختار فایل‌های پروژه

```text
school-report-system/
│
├── main.py
├── auth.py
├── students.py
├── grades.py
├── report.py
├── data.py
├── CONTRIBUTING.md
└── README.md
```

## توضیح فایل‌ها

| فایل              | وظیفه                                      |
| ----------------- | ------------------------------------------ |
| `main.py`         | فایل اصلی برنامه و رابط گرافیکی با Tkinter |
| `auth.py`         | بررسی ورود معلم و دانش‌آموز                |
| `students.py`     | ثبت دانش‌آموز جدید                         |
| `grades.py`       | ثبت نمره و محاسبه معدل                     |
| `report.py`       | تولید متن کارنامه                          |
| `data.py`         | ذخیره و بارگذاری اطلاعات با JSON           |
| `CONTRIBUTING.md` | قوانین همکاری در پروژه                     |

## ترتیب درست انجام کار

در این پروژه هر کار باید با این ترتیب انجام شود:

```text
Issue → Branch → Code → Test → Commit → Push → Pull Request → Review → Merge → Close Issue
```

## قانون مهم

هیچ دانش‌آموزی نباید مستقیم روی شاخه `main` کار کند.

برای هر کار باید:

1.همیشه اول Issue مربوط را بخوانید.
2.سپس Branch مخصوص همان کار را بسازید.
3. کد را بنویسید.
4. برنامه را تست کنید.
5.حتما Commit بزنید.
6.سپس Push کنید.
7. در نهایت Pull Request بسازید با پیام جامع حاوی شماره Issue.
8. منتظر بررسی مربی بمانید.

## اجرای پروژه

برای اجرای پروژه، فایل زیر را در Thonny باز کنید:

```text
main.py
```

یا در ترمینال اجرا کنید:

```bash
python main.py
```

## سناریوی تست نهایی

در پایان پروژه باید این سناریو کار کند:

1. برنامه اجرا شود.
2. معلم با اطلاعات زیر وارد شود:

```text
username: teacher
password: 1234
```

3. معلم دانش‌آموز جدید ثبت کند:

```text
username: ali
password: 1111
```

4. معلم برای دانش‌آموز نمره ثبت کند:

```text
math: 18
science: 19
english: 17
```

5. اطلاعات ذخیره شود.
6. برنامه بسته و دوباره اجرا شود.
7. دانش‌آموز وارد شود:

```text
username: ali
password: 1111
```

8. کارنامه نمایش داده شود.
9. معدل درست محاسبه شود.

معدل مورد انتظار:

```text
18.0
```
